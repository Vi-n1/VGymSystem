# Como usar?
## Siga os passos no terminal

- Clone o repositório
```
git clone https://github.com/Vi-n1/VGymSystem.git
```
- 📁 Entre na pasta VGymSystem
```
cd VGymSystem
```
- ⚙️ Instale as dependências
```
python -m pip install -r requirements.txt
```
- 📁 Entre na pasta src
```
cd src
```
- Inicie o programa
```
python main.py
```
